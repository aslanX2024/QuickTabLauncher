# Security Policy

QuickTabLauncher is a local Windows desktop utility. It does not need API keys, cloud credentials, or network secrets.

Do not commit:

- personal `.lnk` or `.url` files
- private customer/project URLs
- generated `publish` or `publish-portable` folders
- `Notes`
- custom icons that contain private branding
- tokens, passwords, or API keys

If you find a security issue in the launcher behavior, open an issue with a minimal reproduction and avoid posting private paths or credentials.
